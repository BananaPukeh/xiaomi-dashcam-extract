# Xiaomi Dashcam Extractor

This script will extract all files from the Xiaomi 70mai A800 4k Dashcam

This is a Front and Back camera.

## Usage

There are two storage systems

### The dashcam. 

The dashcam uses the following structure

```
/dev/70Mai/
  Event/
    Front/
    Back/
  Normal/
    Front/
    Back/
  Parking/
  Photo/
```

### Your computer

```
dashcam/
  GT86/
    2021-06-05/
      Front/
        ...
      Back/
        ...
  GRS184/
    2021-06-05/
      Front/
        ...
      Back/
        ...
    ...
```
